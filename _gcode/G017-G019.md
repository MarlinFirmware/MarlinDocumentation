Select workspace plane XY, ZX, or YZ. Allows [`G2/G3`](/docs/gcode/G002-G003.html) and [`G5`](/docs/gcode/G005.html) to operate in the selected plane when `CNC_WORKSPACE_PLANES` is enabled.

<h3>Usage</h3>

<span style="display: inline-block; width: 150px;">`G17`</span> Select Plane XY

---
<span style="display: inline-block; width: 150px;">`G18`</span> Select Plane ZX

---
<span style="display: inline-block; width: 150px;">`G19`</span> Select Plane YZ
