Get the current position of the active nozzle. Includes stepper values.

If `M114_DETAIL` is enabled the `D` parameter will provide more details such as leveling information and kinematics.
If `M114_DETAIL` is enabled the `E` parameter will provide E stepper position.
If `M114_REALTIME` is enabled the `R` parameter will report the realtime position instead of projected.
if `M114_LEGACY` is enabled planner.synchronize is called before report_current_position_projected.
