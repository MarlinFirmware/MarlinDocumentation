parameters:
  -
    tag: B
    optional: true
    description: Brightness value (0 - 255)
    values:
      -
        tag: brightness
        type: int

---

Set and/or get the LCD brightness. The value is constrained based on the LCD, but typically a value of 0 is the dimmest and 255 is the brightest.
